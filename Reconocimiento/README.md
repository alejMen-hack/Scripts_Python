Este Script esta en la capacidad de acceder a la información de archivos terminados en .txt, .conf, .env; en busqueda de una palabra clave que se le proporciona para encontrar informacion potencialmente sensible.

La manera de hacer uso de este script es con el siguiente comando: python Reconocimiento_LOcal.py "RUTA" -palabraClave-

En el punto de -RUTA- (esta ruta debe de ir entre comillas) se coloca el directorio en donde se quiere buscar archivos que contengan la palabra clave que se proporciono.
